# How to play
## Snake movement
- Move up: Up arrow key
- Move down: Down arrow key
- Move left: Left arrow key
- Move right: Right arrow key

# Game functionality
- Snake grows a square everytime it eats the food

## Game Over functionality
- If the snake crashes with the wall --> Game Over
- If the snake craches with its own tail -> Game Over

## Score functionality
- The score goes up everytime the snake eats the food
- The highest score gets stored on a file
- The highest score appears on the game everytime the player opens it again
- If the player reached a score higher than the last highest score (if score > highestScore), that new highest score replaces the last one
